Compiling your first project

Clone#

Since we use Git submodules you need to clone them alongside the project. The easiest way is to use the command

git clone --recursive https://github.com/CoolLibs/Demo

If later on while compiling you get an error like

include could not find requested file:  Cool/cmake/IncludeMe.cmake

it most likely means you forgot the submodules. To grab them after having cloned the project, use 

git submodule update --init --recursive

Build#

Install CMake. If you already have it, make sure you have version 3.16 or later.

Then, I recommend this VS Code extension : ms-vscode.cmake-tools.

Finally you will need to setup the extension with a compiler. Here is the tutorial. It is based on Linux but at the bottom of the page you will find the explanations to adapt it for Windows and Mac.
